# Contributing

Thank you for your interest in contributing to the Energa My Meter API integration!

## How to Contribute

1. **Open an issue first** — describe the bug or feature you'd like to work on
2. **Fork** the repository and create a new branch
3. **Make your changes** and ensure they work locally
4. **Submit a Pull Request** with a clear description of the changes

## Requirements

- All PRs must pass the CI checks (`hacs-validate` and `hassfest-validate`)
- Update `README.md` if your changes affect the user-facing interface
- Follow the existing code style

## Reporting Issues

If you encounter a bug, please include:
- Home Assistant version
- Integration version
- Relevant log entries (Settings → System → Logs, filter by `energa`)
